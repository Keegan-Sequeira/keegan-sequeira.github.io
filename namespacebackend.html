Detailed Description

This python module is used to control the backend of the game.

The functions in this module are used to manipulate and create the board, and Tetrominoes It also includes functions that deal with data access for highscores and settings.

Check if a move is valid.

This function will check if a piece can be move by a certain amount of units in any direction. Most of the time this function will be used to check for one moving unit at a time, however, can check for multiple if needed.

Parameters
boardthe board the piece will be moving through
piecethe tetromino object that is moveing through the board
byXhow many units the piece will move along the x-axis
byYhow many units the piece will move along the y-axis
widththe width of the board
heightthe height of the board
Returns
The status of the potential move. 0: Will hit the bottom of board 1: Will hit the sides of the board 2: The bottom of the piece will collide with another piece 3: The side of the piece will collide with another piece 4: No collisions will appear, and the piece is clear to move
